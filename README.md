# Music-Playlist-Generator

## Problem

Many people save songs but do not organize them properly. It becomes difficult to find songs based on mood like happy, sad, or chill.

## Solution

This project is a simple command-line based music playlist generator. It allows users to store songs and generate playlists according to mood. It also supports marking favorite songs and deleting songs.

## Features

* Add new songs with name, artist, and mood
* View all saved songs
* Generate playlist based on mood
* Shuffle playlist automatically
* Delete songs
* View favorite songs

## Technology Used

* Python
* File Handling

## Project Structure

music-playlist-generator/
│── main.py
│── songs.txt
│── README.md

## How to Run the Project

Step 1: Clone the repository
git clone https://github.com/your-username/music-playlist-generator

Step 2: Go to project folder
cd music-playlist-generator

Step 3: Run the program
python main.py

## How to Use

* Run the program
* Choose options from the menu
* Add songs with mood
* Generate playlist by entering mood
* Mark songs as favorite
* Delete songs if needed

## Example

You can add songs like:
Song: Perfect
Artist: Ed Sheeran
Mood: chill

Then generate a playlist by typing "chill".
